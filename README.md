# LASSO 优化算法对比实验

## 📌 项目简介

本仓库实现了 **8种不同的优化算法** 求解LASSO（最小绝对收缩和选择算子）回归问题，针对不同数据维度进行系统性对比分析，为实际应用提供算法选择参考。

## 🎯 实验目标

1. **实现8种优化算法** 求解LASSO问题
2. **对比分析** 三种典型场景下的算法表现：
   - 高维小样本（n < p）
   - 等维数据（n = p）  
   - 低维多样本（n > p）
3. **评估维度**：
   - 计算效率（耗时、迭代次数）
   - 求解精度（目标函数值）
   - 稀疏性表现（零系数占比）

## 📊 算法列表

| 算法类别 | 具体算法 | 核心特点 |
|---------|---------|---------|
| **梯度类算法** | 次梯度下降 | 基础非光滑优化方法 |
| | 近端梯度下降 | 软阈值算子处理L1正则 |
| | 加速梯度下降 | Nesterov加速，收敛快 |
| **逐维度优化** | 坐标下降 | 闭式解，高维场景高效 |
| **分布式算法** | ADMM | 问题拆分，稳定性强 |
| **大规模数据算法** | 随机梯度下降 | 批次计算，内存友好 |
| | 随机方差缩减 | 方差缩减，加速收敛 |
| | Adam | 自适应学习率 |

## 🔬 实验设置

### 数据生成
- **特征矩阵X**：标准正态分布 N(0,1)
- **真实系数β**：10%非零（稀疏），90%为零
- **标签向量y**：y = Xβ + 0.1×N(0,1)，添加高斯噪声

### 三种实验场景
| 场景类型 | (n, p) | 特点 |
|---------|--------|------|
| 高维小样本 | (100, 500) | n < p，LASSO核心应用场景 |
| 等维数据 | (500, 500) | n = p，常规回归场景 |
| 低维多样本 | (1000, 100) | n > p，大样本拟合场景 |

### 统一参数
- 正则化参数：λ = 0.1
- 最大迭代次数：10⁴
- 收敛阈值：10⁻⁶
- 学习率：SGD/Adam=0.001，其他=0.01

## 📈 核心发现

### 场景最优算法
| 场景 | 最优算法 | 次优算法 | 关键指标 |
|------|----------|----------|----------|
| **高维小样本** | **加速梯度下降** | 近端梯度下降 | 耗时最短(0.053s)，稀疏性81% |
| **等维数据** | **坐标下降** | 加速梯度下降 | 仅7次迭代，精度最优 |
| **低维多样本** | **坐标下降** | 加速梯度下降 | 耗时最短(0.026s)，稀疏性91% |

### 重要结论
1. **通用最优**：加速梯度下降在所有场景表现均衡，适合工程应用
2. **场景专用**：坐标下降在低维/等维场景极快，但高维性能下降
3. **稀疏性缺陷**：SGD、Adam等随机梯度类算法无法产生稀疏解
4. **精度妥协**：ADMM牺牲精度换取稳定性，不适合单机LASSO
